void CommonUtilities.TasksManager.AddTask ( Task task)

Adds provided task to pool of managed tasks.

Additionally whenever this method is being invoked, completed tasks are removed from pool of managed tasks. It is simple yet effective mechanism of lazy-management of the pool.
